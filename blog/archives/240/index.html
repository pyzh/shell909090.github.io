Windows Installer修复记录

Oct 20, 2005 - 1 minute read - Comments

贝壳遇到邪门事了，msiexec的服务死活起不来。这个东西服务态啊，又不好做跟踪的。贝壳无奈只好求助于google。结果找到一个解决办法，适用于windowsXPWindows

Installer服务启动即关闭症状的解决。

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINESYSTEMCurrentControlSetSevicesMS IServer]
"ImagePath"=-
"ImagePath"=hex(2):25,00,53,00,79,00,73,00,74,00,65,00,6d,00,52,00,6f,
00,6f,00,74,00,25,00,5c,00,53,00,79,00,73,00,74,00,65,00,6d,00,33,00,32,
00,5c00,6d,73,00,69,00,65,00,78,00,65,00,63,00,2e,00,65,00,78,00,65,00,
20,00,2f,00,56,00,00,00

注意最后是两个回车。

特存留念，方便自己和大家以后使用。
